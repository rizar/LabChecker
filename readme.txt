LabChecker - система для автоматической проверки лабораторных работ по растровой графике.

Исполняемый файл - LabChecker.jar. Руководство по пользованию - LabCheckerDoc.pdf.
Настройки LabChecker - config.xml. Файл не предназначен для ручного редактирования.
В подкаталогах каталога data хранятся лабораторные. Для каждой лабораторной хранятся
        - lab_config.xml - общие сведения о лабораторной работе. Документация - LabConfigDoc.pdf
        - файлы изображений, необходимых для проверки задания.
        - скрипты для проверки шагов лабораторной работы. Документация - ScriptDoc.pdf
